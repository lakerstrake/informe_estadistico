## Reglas
- Formato **JPG** (o PNG). Descarga en la **mayor resolución** disponible.
- Usa bancos **gratuitos y sin atribución**: unsplash.com, pexels.com, pixabay.com.
- Para `treponema.jpg` usa **Wikimedia Commons** (imagen científica); cítala en Referencias.
- Evita imágenes clínicas gráficas: prioriza control prenatal, laboratorio, ciudad y la bacteria.
- Renombra cada archivo EXACTAMENTE como la tabla. Si falta alguna, esa sección simplemente no muestra imagen.
